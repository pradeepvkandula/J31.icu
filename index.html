MAP calculator — standalone, feeds nothing

/ = — mmHg MAP
MAP = (systolic + 2 × diastolic) / 3
Formulas.
BSA (DuBois) = 0.007184 · ht0.725 · wt0.425. Fick CO = VO₂ / (1.34 · Hb · 10 · (SaO₂−SvO₂)). CI = CO / BSA · SV = CO·1000/HR. SVR = 80·(MAP−RAP)/CO dynes, or (MAP−RAP)/CO Wood units. Indexed forms (SVRI, PVRI, TPRI) use CI in place of CO. PVR = 80·TPG/CO dynes = TPG/CO WU. TPG = mPAP−PCWP · DPG = PAd−PCWP. TPR (total pulmonary) = 80·mPAP/CO dynes = mPAP/CO WU. Wood units = dynes ÷ 80. Cardiac power output = MAP·CO/451. PA compliance = SV/(PAs−PAd).
